Snake number pattern in javascript

Snake pattern is pyramid type pattern but numbers flowing in direction resembling to movement of snake.

Code:

n = 5;
var k = (n * (n + 1)) / 2;

for (var i = 0; i < n; i++) {
  row = "";
  for (var j = 0; j <= i; j++) {
    var skip = 0;
    for (m = j - 1; m >= 0; m--) {
      skip = skip + (n - m);
    }
    if (j % 2 == 0) {
      row += k - skip - (i - j) + " ";
    } else {
      row += k - skip - (n - i + (j - 1) - j) + " ";
    }
  }

  console.log(row);
}

Output:

15 
14 7 
13 8 6 
12 9 5 2 
11 10 4 3 1

Above code works for any number `n`, just replace n = 5 with your number.
